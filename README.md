# RoboRams Vision System Fork # </>

# This is a fork of our official FRC Code.

This repository is used for experimentation and nightly development of new features before they are merged
onto the official fork.


## Purpose
- Develop new features in a controlled environment.
- Fine tune features before deploying to our bot's RoboRIO.
- Prototype new autonomous features


## Experimental Features 👨🏻‍💻

- Camera Vision / Aim Assist Pipeline System
- Custom Auto Aim Command
- Shooting while moving with accuracy correction
- Pheonix Experimental Motor Support

- ## Code is unstable and may change frequently and radically.

  ## Requirements
  - WPILib (2026)
  - PhotonVision & dependencies
  - CTRE Pheonix Libraries
